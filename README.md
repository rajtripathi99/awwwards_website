## ğŸš€ Getting Started

### Prerequisites
Node.js installed

### Installation

Clone repository:
```bash
git clone https://github.com/rajtripathi99/awwwards_website.git
```

Navigate:
```bash
cd awwwards_website
```

Install dependencies:
```bash
npm install
```

Start server:
```bash
npm run dev
```

App runs on:
```bash
http://localhost:5173
```
